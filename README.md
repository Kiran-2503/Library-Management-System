# 📚 Library Management System

A terminal-based **Library Management System** developed in **Python** that manages books, users, and library transactions using **CSV files** for permanent data storage.

The application provides separate interfaces for **Library Managers** and **Library Users**, making it easy to manage book inventory and borrowing records through a clean and user-friendly terminal interface.

---

## 🌟 Key Features

- Terminal-based Library Management System
- Separate Manager and User interfaces
- CSV-based permanent data storage
- Automatic database initialization
- Book inventory management
- Borrow and return management
- Transaction history tracking
- Search functionality
- Modular code structure
- Clean terminal user interface

## 👥 User Roles

### Library Manager

The Manager can:

- Login using Manager PIN
- Register new users
- Remove users
- Add new books
- Remove books
- View all books
- Search books
- View transaction history

### Library User

The User can:

- Login using User ID
- View available books
- Borrow books
- Return books
- View personal transaction history

---

## ⚠️ Limitations

- Terminal-based application
- Single Manager PIN authentication
- No password authentication for users
- CSV database is suitable only for small-scale projects
- No fine calculation
- No due date reminder
- Single-user environment
