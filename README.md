# 📚 Library Management System

A terminal-based Library Management System developed using Python. This project allows users to manage library records efficiently through a simple command-line interface.


## Features
- Add new books
- View all books
- Search books by title or ID
- Issue books
- Return books
- Delete books
- Persistent data storage using file handling
- User-friendly menu-driven interface


## Technologies Used
- Python 3
- File Handling
- Command Line Interface (CLI)
